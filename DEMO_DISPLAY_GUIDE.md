# Gear Scavenger 展示指南

## 主菜单按钮

| 按钮 | 作用 |
| --- | --- |
| Story Mode | 标准敌人数量与奖励。完成 Wave 1–3 后进入 Wave 4 Boss。 |
| Challenge Mode | 标准流程基础上，开局立即额外生成一组敌人，适合展示高强度战斗。 |
| Training Mode | 开局额外生成两把随机武器，适合教学与展示不同武器。 |
| START GAME | 使用当前高亮选择的模式开始游戏。 |
| TUTORIAL | 打开操作、技能核心和波次规则说明。 |
| QUIT | 在最终构建程序中退出游戏；在 Unity 编辑器中只记录提示。 |

## Unity 编辑器内放大展示

1. 点击 Unity 下方的 `Game` 标签。
2. 将鼠标放在 Game 窗口上，按 `Shift + Space`，Game 窗口会最大化。
3. 再按一次 `Shift + Space` 恢复编辑器布局。
4. 运行游戏时按 `F11` 可以切换全屏。

## 最终演示程序

1. 在 Unity 顶部菜单点击 `Gear Scavenger > Build Windows Demo`。
2. 等待构建完成。
3. 打开 `Game/Builds/Windows/Gear Scavenger.exe`。

构建程序默认使用 `1920 × 1080` 全屏窗口模式，并允许调整窗口大小。
最终展示推荐直接运行该程序，不展示 Unity 编辑器界面。
