# Mechs Heavy Enemy Candidates

Source: https://marinesosa.itch.io/mech-assets

Planned roles: boss and heavy enemies.

The page includes `Mechs Assets Pack - Free.rar` as a zero-cost download. The
larger full pack is optional and is not required for this project.

The source page permits modification and commercial use, but the asset files
must not be resold or redistributed as an asset pack. Keep the untouched source
package out of a public GitHub repository.

## Preparation

- Select a large, visually dense mech for the boss.
- Use smaller mech bodies for slow heavy enemies.
- Add magenta or hot-red core lights to identify boss attacks.
- Favor projectile, shockwave, and turret attacks because the source currently
  provides idle animation rather than full locomotion animation.
