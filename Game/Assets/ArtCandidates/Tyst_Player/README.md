# Tyst Robot Candidate

Source: https://tyst.itch.io/top-down-robotic-enemies

The source is a zero-cost `Name your own price` download. The creator explicitly
permits commercial use and says credit is appreciated but not required.

Planned role: player scavenger robot. The complete movement and combat animation
sets make this a stronger player candidate than the smaller backup packs.

## Preparation

- Choose a humanoid or mobile robot with readable directional movement.
- Replace enemy-colored highlights with cyan.
- Add asymmetric scavenged armor or a backpack silhouette so the player cannot
  be mistaken for an enemy.
- Preserve the original movement, attack, hurt, and death animation layout.
- Record the author's requested credit in the final credits.

## Suggested output

- `Prepared/player_idle.png`
- `Prepared/player_move.png`
- `Prepared/player_attack.png`
- `Prepared/player_hurt.png`
- `Prepared/player_death.png`

The current game only loads one static `player.png`. Animation integration is a
separate code step after the candidate is approved.
