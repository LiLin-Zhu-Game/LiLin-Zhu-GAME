# Gear Scavenger Known Issues

## Release Scope

- Audio and ambience are intentionally excluded from this release.
- The game currently targets keyboard and mouse on Windows.
- Code-driven animation is used for runtime readability; prepared source packs
  contain additional frame animation that may be integrated in a later release.

## Presentation Notes

- Use Training Mode for the short end-to-end presentation route.
- Use F11 or the built Windows player for a fullscreen presentation.
- Unity Game view performance can differ from the built player while the editor
  is importing assets or displaying multiple panels.

## Reporting

Create a GitHub issue for any blocking error and include:

- Game mode and wave
- Steps to reproduce
- Screenshot or short recording
- Unity Console error text
- Whether the problem also occurs in the Windows build
