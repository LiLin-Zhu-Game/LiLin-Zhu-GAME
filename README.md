# Gear Scavenger

Gear Scavenger is a top-down Unity robot combat game. Clear three generated
machine sectors, retain scavenged weapons and upgrades between Waves, then
destroy three different Boss machines in Wave 4.

## Controls

- `WASD`: move
- `Mouse`: aim and fire
- `Space`: dash
- `E`: compare and equip a nearby weapon
- `Q`: Scrap Nova
- `F`: Magnetic Guard
- `R`: purge weapon heat
- `Esc`: pause or resume
- `F11`: toggle fullscreen

## Modes

- Story: standard enemy counts and progression.
- Challenge: extra enemy pressure.
- Training: reduced room squads and extra weapons for a five-minute full-flow
  rehearsal.

## Build and Release

Open `Game` with Unity `2022.3.62f3c1`, then use:

- `Gear Scavenger > Run Release Readiness Check`
- `Gear Scavenger > Build Windows Demo`

See `RELEASE_PLAYTEST_CHECKLIST.md`, `CREDITS_AND_LICENSES.md`, and
`KNOWN_ISSUES.md` before publishing a build.
